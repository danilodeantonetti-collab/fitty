# Fitty \u2013 Pro Workout Tracker \ud83d\udcaa

A premium Progressive Web App for tracking full-body workouts, built with Next.js 16, Supabase, and Tailwind CSS 4.

## Features

- \ud83c\udfcb\ufe0f Workout Plans (Tag A & Tag B)
- \ud83d\udcca Statistics Dashboard with volume tracking
- \ud83d\udcc5 Date picker for session logging
- \ud83d\udcf1 PWA \u2013 installable on Android & iOS
- \ud83d\udd12 Supabase Auth + Row Level Security
- \ud83c\udf19 Dark mode with neon accent design

## Getting Started

```bash
npm install
npm run dev
```

Create a `.env.local` with your Supabase credentials:

```
NEXT_PUBLIC_SUPABASE_URL=your_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
```

## Tech Stack

- **Next.js 16** \u2013 React framework
- **Supabase** \u2013 Auth + PostgreSQL
- **Tailwind CSS 4** \u2013 Styling
- **PWA** \u2013 Service Worker + Manifest
